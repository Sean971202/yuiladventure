關於我們

我們是一群在花蓮玉里高中資料處理科的學生。這裡不只是我們的家鄉，更是一座等待被重新定義的數位寶藏。

「玉里奇遇」專題的初衷，是希望能打破傳統文化紀錄的隔閡。我們透過程式架構，試圖將玉里鎮上的歷史脈絡、在地人文與景點故事，轉化為一場觸手可及的數位冒險。

這不僅是一個專題，更是我們獻給家鄉的一封數位情書。我們相信，科技的價值，在於讓被遺忘的歷史與美好角落，重新在旅人的螢幕上閃閃發光。

創作故事

這場旅程從零開始。當我們在開發初期，面對的是無數的程式邏輯難題，以及如何將厚重的歷史文化轉化為生動的互動遊戲。

我們將編碼視為畫筆，以 Python 與 Java 構築出遊戲的核心架構，確保每一個景點的數據能精準對應到互動環節；我們將攝影視為載體，帶著 Sony a6600 走遍玉里的每一條街道，捕捉客城紅橋下那道劃過金黃稻浪的列車身影。

從最初的一張草稿圖，到現在這個能夠與使用者互動的系統，過程充滿了實驗與修正。我們在程式碼的括號間磨練邏輯，在影像的快門聲中深化對家鄉的情感。這段經歷教會我們，技術不只是冷冰冰的數據，當它與土地連結時，便擁有改變觀點的力量。

聯絡與開發回饋
