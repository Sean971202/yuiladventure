關於我們

我們是一群在花蓮玉里成長、學習的學生。這裡不只是我們的家鄉，更是一座等待被重新定義的數位寶藏。

「玉里奇遇」專題的初衷，是希望能打破傳統文化紀錄的隔閡。我們透過 Leaflet.js 與 Java 軟體架構，試圖將玉里鎮上的歷史脈絡、在地人文與景點故事，轉化為一場觸手可及的數位冒險。

這不僅是一個專題，更是我們獻給家鄉的一封數位情書。

創作故事

我們將編碼視為畫筆，以 Python 與 Java 構築出遊戲的核心架構；我們將攝影視為載體，帶著 Sony a6600 走遍玉里的每一條街道，捕捉客城紅橋下那道劃過金黃稻浪的列車身影。

從最初的一張草稿圖，到現在能夠與使用者互動的系統，這段經歷教會我們，技術不只是冷冰冰的數據，當它與土地連結時，便擁有改變觀點的力量。

聯絡與開發回饋

你的每一個意見，都是讓「玉里奇遇」成長的養分。協助我們改善互動體驗，讓這份作品能更貼近每一位旅人的需求。
